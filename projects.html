4-Bit LED Binary Counter

What I Built

Created a simple but fundamental embedded systems project using an Arduino to control 4 LEDs that display numbers 0-15 in binary format. The counter increments automatically, showing how digital systems represent numbers at the hardware level.

What I Learned

  • Binary number systems - How computers represent data using only 1s and 0s
  • Arduino programming - digitalWrite(), delay(), loop structures, and pin configuration
  • Circuit design - Breadboard layout, LED current limiting with resistors, proper grounding
  • Embedded systems thinking - How software controls physical hardware in real-time
  • Bitwise operations - Converting decimal numbers to binary representation programmatically

Technical Details

Used 4 digital output pins, implemented bit shifting to extract individual binary digits, and created a visual representation of how data is stored in computer memory. This project bridges the gap between abstract programming concepts and tangible hardware results.

Instagram Media Archiver

[Image placeholder - Add code snippet or diagram of scraping logic]

What I Built

Developed a Python script to automatically download and archive saved Instagram videos to a personal server. The goal was to create a permanent backup of content that might otherwise disappear when posts are deleted.

What I Learned

  • Web scraping - How to programmatically extract data from websites
  • API reverse engineering - Understanding how Instagram's internal APIs work
  • Rate limiting - Why platforms implement request throttling and how to detect it
  • HTTP requests - Authentication, session management, and request headers
  • File system operations - Downloading, storing, and organizing media files programmatically
  • Ethical considerations - The line between personal data backup and ToS violations

Outcome & Reflection

While Instagram flagged the automated activity before I could complete the full archive, the technical implementation worked as intended. This taught me about the challenges of working with third-party platforms, the importance of respecting rate limits, and how companies protect their data. The project demonstrated I can build working solutions to real problems, even if external constraints prevent full deployment.
